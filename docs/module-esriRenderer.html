Methods

(inner) getMetadata(layer)

Start the process of adding a vector layer to the map.
uses the mapJson-layer object to get the metadata of the ArcGIS Server layer.
After the metadat is obtained the process continues through either parseSimpleRenderer or parseUniqueValueRenderer depending on renderer type.
Adds the following metadata to the layer:
  1. "metadata": fields
  2. "geomType": geometry type ("esriGeometryPoint"|"esriGeometryPolygon"|"esriGeometryPolyline")
  3. "labelingInfo" : esri labeling info object
  4. "renderer": full renderer object with rendering instructions Still missing: parsing class breaks symbology
Parameters:
Name Type Description
layer mapJson-layer An object of mapJson-layer type

(inner) updateRaster(sourceName)

Update the image for an image source.
based on map bounds when the function is called, a new image is requested from the ArcGIS Server using the export endpoint.
The update is done using `map.getSource(sourceName).updateImage({ url: curUrl, coordinates: coords})` which needs both current map extent for coordinates and a new image.
Parameters:
Name Type Description
sourceName String name of the source, as added to map context
